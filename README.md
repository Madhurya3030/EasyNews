# 📰 EasyNews – AI Powered Interactive News Platform

## 🚀 Overview
EasyNews is an AI-powered news platform designed to simplify how users consume news. It provides personalized, interactive, and easy-to-understand content using modern web technologies and Natural Language Processing (NLP).

---

## ✨ Features
- 🔍 **Personalized News Recommendations** based on user interests and activity  
- 🧠 **AI-Based Summarization** to convert long articles into concise insights  
- 🤖 **Chatbot Integration** for interactive explanations  
- 🔊 **Multi-format Content** (Text, Audio, Video)  
- 🌐 **Multi-language Support** for wider accessibility  
- 📈 **Trending News Detection** using engagement analysis  
- ⚡ **Real-time News Fetching** via external APIs  

---

## 🛠️ Tech Stack
**Frontend:** React.js  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  

**AI / NLP:**
- TF-IDF (content-based recommendation)
- NLP techniques for summarization

**APIs:**
- News API (real-time data)

---

## 🧩 How It Works
1. Fetches real-time news using APIs  
2. Processes articles using TF-IDF and NLP  
3. Matches content with user preferences  
4. Generates summaries and insights  
5. Displays content in multiple formats
